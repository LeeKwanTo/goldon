### 钢筋算量 答题流程 ###

①先做钢筋汇总算量-->保存-->交卷

②再做土建-->导入钢筋工程-->套外部清单-->补画其他构件（套外部清单）-->汇总计算-->保存-->交卷

模考和正赛都可以**多次交卷**，钢筋先交卷不影响成绩导入到土建

钢筋推荐导入的构件：**轴网、柱、梁、墙**

手动建立的构件：**楼层板、板、板筋、筏板基础、筏板钢筋**
